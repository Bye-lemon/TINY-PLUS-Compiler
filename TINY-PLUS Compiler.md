# TINY-PLUS Compiler

#### 文法的定义

| 保留字 | Token  |
| ------ | ------ |
| if     | IF     |
| then   | THEN   |
| else   | ELSE   |
| repeat | REPEAT |
| until  | UNTIL  |
| read   | READ   |
| write  | WRITE  |
| end    | END    |

| 运算符 | Token  |
| ------ | ------ |
| +      | PLUS   |
| -      | MINUS  |
| *      | TIMES  |
| /      | OVER   |
| =      | EQ     |
| <>     | NOTEQ  |
| <      | LT     |
| >      | GT     |
| <=     | LTEQ   |
| \>=    | GTEQ   |
| (      | LPAREN |
| )      | RPAREN |
| ;      | SEMI   |
| :=     | ASSIGN |

| 标识符 | Token |
| ------ | ----- |
| 数字   | NUM   |
| 标识符 | ID    |
